SUS Calculator adalah aplikasi berbasis web sederhana yang dirancang untuk membantu praktisi UX (User Experience) dalam menghitung skor System Usability Scale (SUS) secara cepat dan akurat. Proyek ini dibangun menggunakan framework Laravel 11 dan Tailwind CSS.

Fitur Utama
Kalkulasi Otomatis: Mengimplementasikan logika perhitungan SUS (perbedaan rumus untuk pertanyaan ganjil dan genap).

Interpretasi Hasil: Memberikan hasil akhir berupa skor (0-100) beserta Grade (A-F) dan Adjective Rating (Excellent, Good, Poor).

Desain Responsif: Tampilan yang bersih dan modern menggunakan Tailwind CSS, nyaman diakses melalui perangkat mobile maupun desktop.

Input Validasi: Memastikan semua data terisi dengan benar sebelum perhitungan dilakukan.

Tech Stack
Backend: Laravel 11 (PHP 8.2+)

Frontend: Blade Template Engine & Tailwind CSS

Tools: Composer, Artisan.
